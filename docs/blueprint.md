# **App Name**: David Sokoya Portfolio

## Core Features:

- Profile Showcase: Display personal and professional details including name, title, bio, and contact information.
- Skills Display: Showcase technical skills, including languages, frameworks, databases, and tools.
- Project Portfolio: Display detailed project descriptions with live demo URLs, GitHub repo links, screenshots, and descriptions of challenges and solutions.
- Real-world Work: Highlight freelance work and volunteer projects with descriptions and links.
- Job Description Link: Display a link to a job description for reference and inspiration, signaling active job search goals.

## Style Guidelines:

- Primary color: Light, muted blue (#A7C4BC) to evoke a sense of calm and professionalism, in alignment with a minimalist style.
- Background color: Very light grey (#F0F0F0) to maintain a clean, minimalist aesthetic.
- Accent color: Soft olive green (#9E9D5E) as an analogous contrast to the primary color, without clashing with the minimalism.
- Body and headline font: 'Inter' (sans-serif) for a clean, modern, and readable design.
- Minimalist layout with clear sections for easy navigation and readability. Ample white space to reduce clutter and emphasize content.
- Simple, line-based icons to represent different skills and sections. Consistent style to match the minimalist aesthetic.
- Subtle transition effects for navigation and content loading to enhance user experience without being distracting.
